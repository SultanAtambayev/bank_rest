# Система управления банковскими картами

## Описание проекта
REST API для управления банковскими картами с JWT аутентификацией, ролевой моделью (USER/ADMIN) и переводами между картами.

## Архитектура
Проект построен по многослойной архитектуре:
- Controller - обработка HTTP запросов
- Service - бизнес-логика
- Repository - работа с базой данных
- Entity - JPA сущности
- DTO - объекты передачи данных
- Security - JWT аутентификация
- Exception - обработка ошибок
- Util - утилиты (шифрование)

## Технологии
- Java 21
- Spring Boot 3.2.5
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL
- Liquibase
- Docker Compose
- Swagger/OpenAPI
- JUnit 5 + Mockito

## Функциональность

### Пользователь (USER)
- Регистрация и аутентификация
- Создание карт
- Просмотр своих карт (пагинация)
- Блокировка своих карт
- Переводы между своими картами
- Просмотр полного номера своей карты

### Администратор (ADMIN)
- Все возможности пользователя
- Просмотр всех пользователей
- Просмотр всех карт (пагинация)
- Активация и блокировка любых карт
- Удаление карт и пользователей
- Изменение ролей пользователей

## Безопасность
- JWT токены (время жизни 1 час)
- Шифрование номеров карт в БД (AES-128)
- Маскирование номеров карт при выводе (**** **** **** 1234)
- Ролевой доступ к эндпоинтам
